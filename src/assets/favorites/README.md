# ⭐ Favorites — drop images here

Put any photos you want in the **"Favorites"** section into **this folder**
(`src/assets/favorites/`).

That's the whole setup. Every image here appears **automatically** — no code to
edit, no captions to fill in. They're shown in filename order.

### Controlling the order
Images are sorted by filename (numerically), so name them like:

```
01-beach.jpg
02-dinner.jpg
03-sunset.jpg
```

### Supported formats
`.jpg` `.jpeg` `.png` `.webp` `.gif` `.avif`

### Tips
- Keep the long edge around ~1200–1600px for sharp, fast-loading photos.
- Lowercase filenames with no spaces are safest (e.g. `movie-night.jpg`).
- After adding/removing images, the dev server picks them up on reload.

> Note: this is a different folder from `public/photos/` (which feeds the
> "Our Favorite Memories" gallery). This one is just for the **Favorites**
> section and requires no editing.
